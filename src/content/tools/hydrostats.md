The HydroStats app is a user interface for the Hydrostats Python package, which provides tools for visualization and validation of hydrological time series data.

## How it works

Hydrostats is a library of tools for working with time series, with functions specific to hydrology. It offers a high-level interface for common tasks such as comparing observed and simulated discharge and computing goodness-of-fit metrics.
