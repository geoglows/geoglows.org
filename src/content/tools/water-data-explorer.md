The Tethys Water Data Explorer (WDE) is a web-based tool that lets users discover, access, visualize, and download data from any information system that publishes water data in WaterML format through WaterOneFlow services.

## About the app

It builds on the CUAHSI Hydrologic Information System (HIS) and standardized WaterOneFlow web services to make sharing water data on the web consistent and accessible.
