Met Data Explorer (MDE) is a web-based tool that lets a broad range of users discover, access, visualize, and download data from any THREDDS Data Server (TDS) that stores meteorological data.

## About the tool

MDE is an open-source application for visualizing meteorological gridded data. It can be customized for local or regional portals, installed by any organization, and requires minimal server space.
